# Vulcan
Vulcan, a Discord application

### Dependancies
- node.js
- discord.js
- dotenv

## Installation

Install node.js from nodejs.org. <br>
Install discord.js by typing `npm install discord.js`<br>
Install Dotenv by `npm install dotenv`<br>

Make a `.env` file and write
```
Token = "{your-token}"
```
